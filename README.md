# MotoCom

## Description

* Android App for close communication via HotSpot or WiFi mainly used when there is no internet.
* Useful for Motobike bluetooth comunication.
* There is no PTT (Push To Talk) so it is like phone call.
* App is in Beta

### Requirements
* Android 7.0+ (API 24+)
